Comparación OSI vs TCP/IP

Modelo OSI

El modelo Open Systems Interconnection (OSI) es un modelo conceptual creado por la Organización Internacional para la Estandarización, el cual permite que diversos sistemas de comunicación se conecten usando protocolos estándar.

Capa 1: Física

Define las caracteristicas fisicas de red material

Capa 2: Enlace de datos

Proporciona servicio de envio de datos a traves del enlace fisico.

Capa 3: Red

Gestiona conexiones a traves de la red para las capas superiores

Capa 4: Transporte

Proporciona servicios de deteccion y correcion de errores

Capa 5: Sesión

Gestiona las conexiones entre aplicaciones cooperativas

Capa 6: Presentación

Estandariza la forma en que se presentan los datos a las aplicaciones

Capa 7: Aplicación

Programas de aplicacion que usan la red

Modelo TCP/IP

El modelo TCP/IP es usado para comunicaciones en redes y, como todo protocolo, describe un conjunto de guías generales de operación para permitir que un equipo pueda comunicarse en una red. TCP/IP provee conectividad de extremo a extremo especificando cómo los datos deberían ser formateados, direccionados, transmitidos, enrutados y recibidos por el destinatario.

Capa de Acceso a Red

Controla los dispositivos del hardware y los medios que forman la red

Capa de Internet

Determina la mejor ruta a traves de la red

Capa de Transporte

Admite la comunicadion entre distintos dispositivos de distintas redes

Capa de Aplicación

Representa datos para el ususario mas el control de codificacion y de dialogo
